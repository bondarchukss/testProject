Цель
Познакомится с стеком технологий Moya, CellVIew, SnapKit, git
Описание
Требуется создать генератор надписей на изображении. Состоит из 2-х экранов.
1 экран: лента картинок (API https://rickandmortyapi.com/api) - при старте запрашивает данные из сети. 
При выборе картинки идет переход на детальный экран с полноразмерной картинкой. На экране можно ввести текст в textview. В нав баре кнопка - поделится. С помощью нее экспортирует картинка с совмещенным текстом.
Стек
Swift 4,1+, RxSwift, Moya, CollectionView or TableView для первого экрана, git, SnapKit
